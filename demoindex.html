Standalone local HTML (this file)

This page will attempt to load the Ringg embed and show a visible error if the embed fails (for example, blocked by a server-side whitelist).

How to run

  1. Save this file as index.html in a folder.
  2. Run a local server (recommended): python -m http.server 8000 then open http://localhost:8000.
  3. If you deploy to GitHub Pages or any public host, ensure that host is added to your Ringg assistant's Whitelist Domains.
